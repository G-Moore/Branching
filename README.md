## Git Branching

## Basic Git Commands

* 'git init' - Create local repo
* 'git add' - Add files
* 'git commit' - Commit to local
* 'git status' - See status of local repo
* 'git log' - View commit log
* 'git log -- oneline' - show short long

## Basic Branching

* 'git branch' - Show branches, current branch
